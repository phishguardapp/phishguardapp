‚ùì Frequently Asked Questions

Is PhishGuard really free?

Yes! PhishGuard is completely free with no subscriptions, in-app purchases, or hidden costs. We believe everyone deserves protection from phishing attacks.

Does PhishGuard track my browsing?

Absolutely not. All analysis happens on your device. We don't collect, store, or transmit your browsing history or personal information. Your privacy is guaranteed.

How accurate is the detection?

PhishGuard uses multiple detection layers for high accuracy:

  • Domain age verification (catches new scam sites)
  • SSL certificate validation (110+ trusted authorities)
  • AI-powered content analysis (on compatible devices)
  • Form security checks
  • Keyword and pattern detection
  • Brand impersonation detection

While no system is 100% perfect, PhishGuard provides excellent protection.

Will it slow down my browsing?

No! PhishGuard uses smart caching and efficient algorithms. Analysis typically takes less than 3 seconds on first visit, and cached results load instantly.

Does it work offline?

Partially. PhishGuard can use cached results offline. New site analysis requires internet for domain age verification, but the extension will still perform local checks.

What if a legitimate site is flagged?

False positives can occur, especially with very new legitimate sites. You can mark the site as trusted by clicking the PhishGuard icon and toggling "Trust this site". Please also report false positives so we can improve.

Which languages are supported?

PhishGuard is available in 7 languages: English, Spanish, French, German, Chinese (Simplified), Japanese, and Hindi. The interface automatically matches your device language.

Can I use it on multiple devices?

Yes! Install PhishGuard on all your iOS and macOS devices. Each installation is independent and free.

üîß Troubleshooting

Badge not appearing in Safari

Solutions:

  • Check that PhishGuard is enabled in Safari settings
  • Restart Safari
  • Reinstall the extension
  • Update to the latest version of Safari and iOS/macOS

Always shows "Unknown" verdict

Solutions:

  • Check your internet connection
  • Try clicking "Rescan" in the popup
  • Wait a few seconds - analysis may still be running
  • Check if WHOIS services are accessible from your network

Legitimate site flagged as suspicious

Common reasons:

  • Very new domain (legitimate startups may be flagged)
  • Uncommon SSL certificate provider
  • Site hosted on shared platform

Solution: Click the PhishGuard icon and toggle "Trust this site" if you're certain it's legitimate. Please report the false positive.

Pages loading slowly

Solutions:

  • PhishGuard uses caching - slowness should only occur on first visit
  • Check your internet connection speed
  • Try disabling and re-enabling the extension
  • Clear Safari cache and restart

Extension not working after update

Solutions:

  • Restart Safari
  • Disable and re-enable the extension in Safari settings
  • Restart your device
  • Reinstall PhishGuard if issues persist

‚úÖ Best Practices for Online Safety

General Security

  • Trust PhishGuard's verdicts - take warnings seriously
  • Always verify URLs before entering sensitive information
  • Use strong, unique passwords for each account
  • Enable two-factor authentication when available
  • Keep your devices and software updated

When Banking Online

  • Bookmark your bank's website and always use the bookmark
  • Never click banking links in emails
  • Verify the URL is correct (check for misspellings)
  • Your bank should always show a green badge
  • Enable all available security features

When Shopping Online

  • Research new stores before purchasing
  • Check domain age - very new stores are risky
  • Use credit cards (better fraud protection than debit)
  • Look for contact information and return policies
  • If prices seem too good to be true, they probably are
